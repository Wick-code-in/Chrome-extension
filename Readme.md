# Exam Upload Assistant

## Overview

This repository contains a private Chrome Extension that assists with uploading multiple-choice questions to an internal company website.

This project follows a phased implementation strategy.

## Documentation

Before writing any code, read these files in order:

1. IMPLEMENTATION_PLAN.md
2. PROJECT_CONTEXT.md
3. DEVELOPMENT_RULES.md

These files together define the project specification.

## Development Workflow

Every implementation session follows this process:

1. Read all project documentation.
2. Implement only the requested phase.
3. Explain the implementation.
4. Provide a manual testing checklist.
5. Stop and wait for approval.

Never continue automatically.
